# Project Resources and Deliverables

This page collects major project resources, design artifacts, research materials, and non-code deliverables created during the DeVILSona capstone. It is intended to provide future teams with quick access to important external materials that support the project but are not stored directly as implementation code in the repository.

## Figma Prototype

The Figma prototype contains the primary design and interaction mockups created for the DeVILSona project. It was used to plan scenario structure, interface flow, and early user experience decisions before and during implementation.

- [Figma Prototype](https://www.figma.com/design/Bdi6wjxaKxjFydbq6f0LLl/FSE100-VR-Customer-Archetypes-Updated-Sprint-2?node-id=0-1&t=xmrnW6LKMITSr4U1-1)

## Customer Archetype Resource Folder

This folder contains the customer archetype reference materials originally created and provided by the project sponsor as part of the project definition. These materials served as the foundation for the team’s implementation of the current DeVILSona experience, including the Military Mike archetype used throughout the existing game.

These resources are also important for future teams, since additional scenarios may be built around other customer archetypes provided by the project sponsor.

- [Customer Archetype Resource Folder](https://drive.google.com/drive/folders/1Py_yuASVd7RmK579FrZ2szgvLpwQZo_4?usp=drive_link)

## Military Veteran Mike Research and Design Materials

These resources document the research, constraints, scenario design work, and scripted conversational material developed specifically for the **Military Veteran Mike** archetype. Together, they form the main non-code design and research foundation for the current DeVILSona implementation.

### Designed Scenarios

This document contains the scenario designs created for the Military Veteran Mike implementation, including scenario descriptions, context, structure, intended interactions, and other planning details used to guide development.

- [Military Veteran Mike Designed Scenarios](https://docs.google.com/document/d/1AjrdD2VNTBt4Jt36pD0bas2lQXylwbtCsCMM8yzugq8/edit?usp=sharing)
