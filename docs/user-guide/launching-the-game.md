# Launching the Game

This guide provides instructions on how to start DeVILSona depending on which version of the game you are using. You can run the game either through a Windows PC (using Meta Quest Link) or directly from your Meta Quest headset (using the installed APK).

---

## 1. Launching with Windows (Desktop Version)
This is the recommended method for the most stable and high-quality experience, particularly for complex scenarios like Scenario 2.

**Prerequisites:**
* A VR-ready Windows PC with the DeVILSona application downloaded.
* The **Meta Quest Link** desktop application installed and running.
* Your Meta Quest headset connected to the PC via a high-quality Link cable or Air Link.

**Steps:**
1. Connect your Meta Quest headset to your PC.
2. Put on the headset and enable **Quest Link** from the quick settings menu to view your desktop interface.
3. On your PC (or via the virtual desktop in VR), navigate to the folder where DeVILSona is installed.
4. Double-click the `DeVILSona.exe` file to launch the game.
5. The game should automatically detect the headset and display the VR environment.

---

## 2. Launching with Installed APK (Standalone Version)
If the game has been sideloaded directly onto your Meta Quest headset, you can launch it without being tethered to a PC.

**Prerequisites:**
* The DeVILSona APK must be properly installed (sideloaded) on your headset.
* The headset must be in **Developer Mode**.

**Steps:**
1. Put on your Meta Quest headset.
2. Open the **App Library**.
3. At the top of the App Library, click on the search bar or the filter dropdown menu.
4. Scroll to the bottom of the list and select **Unknown Sources**.
5. Find **DeVILSona** in the list of applications and select it to launch the game.

> ⚠️ **Note:** Some features or scenarios may experience performance issues or long loading times in the APK version. For the best experience, the Windows desktop version is highly recommended.
