## Troubleshooting: Interview Audio and Microphone Issues

If the interview is not responding to your speech, or if you cannot hear AI responses, check the following issues first.

### 1. Make sure your headset microphone is not muted

On Meta Quest, you can check your microphone status from **Quick controls**.

**Steps to check and unmute the microphone:**

1. Press the **Meta button** on the **right controller**.
2. Open **Quick controls**.
3. Hover over the **microphone icon**.
4. Check whether the microphone is currently muted or unmuted:
   - If the microphone icon has a **slash through it**, the microphone is **muted**.
   - If the microphone icon has **no slash**, the microphone is **unmuted**.
5. Click the microphone icon to toggle it if needed.

If your microphone is muted, the interview may not detect your speech until it is unmuted.

### 2. Check app microphone permissions on the Meta Quest headset (APK version only)

This step applies to the **APK version installed directly on the headset**. It does **not** apply to the desktop application version played through **Meta Quest Link**.

If microphone access is not allowed for the APK app, the interview system may not be able to hear you.

**Steps to review app permissions for the APK version:**

1. Open **Settings** on the Meta Quest headset.
2. Go to **Privacy and safety**.
3. Open **App permissions**.
4. Find the installed application you are using.
5. Make sure **Microphone** permission is enabled for that app.

### 3. Restart the game if AI responses are missing

If you can speak but do not hear AI responses, fully exit the game and restart it. In some cases, restarting the application can restore audio behavior.

This can help in both:
- the **desktop application version through Meta Quest Link**
- the **APK version**

### 4. If problems continue

If the interview still does not work as expected after checking microphone mute status and, when applicable, app permissions:

- close the game completely and reopen it,
- confirm the headset microphone is unmuted,
- if using the **APK version**, confirm the app still has microphone permission enabled,
- then try the interview again.

If the issue still persists after that, additional setup or device-specific troubleshooting may be required.
